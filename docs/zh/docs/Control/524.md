# 5.24 滚动条控件

类名：SScrollBar
控件名：scrollbar
基类：SWindow
说明：滚动条控件
属性：
属性名 类型 说明
skin 字符串 皮肤
arrowSize 数字 箭头大小
min 数字 最小值
max 数字 最大值
value 数字 当前值
page 数字 翻页大小
vertical 0|1 是否垂直（0-- 否 1--是）
